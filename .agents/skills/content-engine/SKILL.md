# 콘텐츠 엔진 (Content Engine)

똑같은 내용을 모든 곳에 똑같이 게시하는 대신, 하나의 아이디어를 각 플랫폼의 특성에 맞는 강력한 콘텐츠로 변환합니다.

## 활성화 시기

- X(트위터) 게시물 또는 스레드를 작성할 때
- LinkedIn 게시물 또는 출시 업데이트 초안을 작성할 때
- 숏폼 영상 또는 YouTube 설명 영상 스크립트를 작성할 때
- 아티클, 팟캐스트, 데모 또는 문서를 소셜 콘텐츠로 재가공할 때
- 제품 출시, 마일스톤 또는 특정 테마를 중심으로 가벼운 콘텐츠 계획을 수립할 때

## 기본 고려 사항

다음 사항을 명확히 하십시오:
- 소스 에셋(Source asset): 무엇을 기반으로 변형할 것인가
- 대상 독자: 개발자, 투자자, 고객, 운영자 또는 일반 대중
- 플랫폼: X, LinkedIn, TikTok, YouTube, 뉴스레터 또는 멀티 플랫폼
- 목표: 인지도 확산, 전환(Conversion), 채용, 권위 확보, 출시 지원 또는 참여 유도

## 핵심 규칙

1. 플랫폼에 맞게 조정하십시오. 동일한 문구를 여러 곳에 그대로 복사해서 게시하지 마십시오.
2. 요약보다 후킹(Hook) 포인트가 더 중요합니다.
3. 모든 게시물에는 하나의 명확한 아이디어만 담아야 합니다.
4. 슬로건보다는 구체적인 내용을 사용하십시오.
5. 요청 사항(CTA)은 가볍고 명확하게 유지하십시오.

## 플랫폼별 가이드라인

### X (트위터)
- 빠르게 시작하십시오.
- 게시물 하나 또는 스레드의 개별 트윗 하나당 하나의 아이디어만 담으십시오.
- 꼭 필요한 경우가 아니면 본문에 링크를 포함하지 마십시오.
- 해시태그 스팸을 피하십시오.

### LinkedIn
- 첫 번째 문장을 강렬하게 작성하십시오.
- 문단을 짧게 유지하십시오.
- 교훈, 결과 및 시사점을 더 명확하게 프레임화하십시오.

### TikTok / 숏폼 영상
- 처음 3초 이내에 시선을 사로잡아야 합니다.
- 단순히 나레이션만 하는 것이 아니라, 시각적인 요소를 고려하여 스크립트를 작성하십시오.
- 하나의 데모, 하나의 주장, 하나의 요청 사항(CTA)만 담으십시오.

### YouTube
- 결과를 초반에 보여주십시오.
- 챕터별로 구조를 잡으십시오.
- 20~30초마다 시각적인 변화를 주십시오.

### 뉴스레터
- 관련 없는 항목들을 모아놓는 것이 아니라, 하나의 명확한 관점을 전달하십시오.
- 섹션 제목을 훑어보기 쉽게 만드십시오.
- 도입 문구부터 실제 가치가 전달되도록 하십시오.

## 콘텐츠 재가공(Repurposing) 흐름

기본 단계:
1. 앵커 에셋(Anchor asset) 선정: 아티클, 영상, 데모, 메모 또는 출시 문서
2. 3~7개의 핵심 아이디어(Atomic ideas) 추출
3. 플랫폼별 네이티브 변체 작성
4. 출력물 간의 반복 내용 제거
5. 플랫폼의 특성에 맞춰 요청 사항(CTA) 조정

## 결과물 구성

캠페인 작성을 요청받은 경우 다음을 반환하십시오:
- 핵심 관점(Core angle)
- 플랫폼별 초안
- 권장 게시 순서 (선택 사항)
- 다양한 버전의 요청 사항(CTA) (선택 사항)
- 발행 전 추가로 필요한 정보

## 품질 검사 (Quality Gate)

제출 전 확인 사항:
- 각 초안이 해당 플랫폼에 어색하지 않게(Natively) 작성되었는가
- 후킹 포인트가 강력하고 구체적인가
- 상투적이고 과장된 표현이 없는가
- 요청받지 않은 경우, 플랫폼 간에 동일한 문구가 중복되지 않는가
- 요청 사항(CTA)이 콘텐츠 및 독자와 잘 어울리는가
