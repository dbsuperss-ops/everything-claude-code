# 참조 출처 및 라이선스 고지

ECC v1.8.0은 다음 프로젝트와 연구로부터 영감을 받았습니다:

- `plankton`
- `ralphinho`
- `infinite-agentic-loop`
- `continuous-claude`
- 공개 프로필: [zarazhangrui](https://github.com/zarazhangrui), [humanplane](https://github.com/humanplane)

## 정책

1. 라이선스가 없거나 호환되지 않는 소스로부터 직접적인 코드를 복사하지 않습니다.
2. ECC 구현 방식은 본 저장소의 아키텍처와 라이선스 모델에 맞춰 새롭게 작성되었습니다.
3. 명시적으로 재사용이 허용되지 않는 한, 참조된 자료는 아이디어, 패턴 및 개념적 프레임워크 수준에서만 활용합니다.
4. 향후 직접적인 재사용이 필요한 경우, 명시적인 라이선스 확인과 함께 파일 내부 및 릴리스 노트에 출처를 표기합니다.
