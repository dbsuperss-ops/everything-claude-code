# X 스레드 초안 - ECC v1.8.0

1/ ECC v1.8.0이 출시되었습니다. 이번 릴리스의 핵심은 단 하나입니다: 더 나은 에이전트 하네스 성능(Agent harness performance).

2/ 후크 신뢰성 개선, 루프 운영 명령어, 그리고 더욱 강화된 평가 워크플로우를 포함하고 있습니다.

3/ NanoClaw v2는 이제 모델 라우팅, 스킬 핫로드(Hot-load), 브랜칭, 검색, 압축, 익스포트 및 지표를 지원합니다.

4/ 에이전트 성능이 떨어진다면, 먼저 `/harness-audit`를 실행하고 품질 게이트(Quality gates)를 강화해 보십시오.

5/ Claude Code, Cursor, OpenCode, Codex 전반에 걸친 크로스 하네스 호환성은 여전히 저희의 최우선 순위입니다.
