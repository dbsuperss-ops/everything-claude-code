# LinkedIn 초안 - ECC v1.8.0

ECC v1.8.0은 이제 시스템 수준에서의 하네스 성능(harness performance)에 집중합니다.

이번 릴리스의 개선 사항:
- 후크 신뢰성 및 라이프사이클 동작 개선
- 평가 주도 엔지니어링(Eval-driven engineering) 워크플로우 강화
- 자율 루프(Autonomous loops) 운영을 위한 도구 추가
- Claude Code, Cursor, OpenCode, Codex 전반에 걸친 크로스 플랫폼 지원

또한 실제 워크플로우에서 강력한 세션 운영을 지원하는 NanoClaw v2가 포함되었습니다.

AI 코딩 워크플로우가 불안정하다고 느껴진다면, 에이전트 하네스를 하나의 독립적인 엔지니어링 시스템으로 대우하는 것부터 시작해 보십시오.
