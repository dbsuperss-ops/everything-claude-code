# 콘텐츠 엔진

하나의 아이디어를 플랫폼에 최적화된 강력한 원본 콘텐츠로 변환합니다. 모든 곳에 똑같은 내용을 복사해서 게시하지 마십시오.

## 적용 시점

* X(트위터) 포스트나 스레드를 작성할 때
* LinkedIn 포스트 초안을 작성하거나 업데이트를 게시할 때
* 숏폼 영상이나 YouTube 설명 스크립트를 작성할 때
* 기사, 팟캐스트, 프레젠테이션 또는 문서를 소셜 콘텐츠로 재가공할 때
* 제품 출시, 주요 마일스톤 또는 특정 주제를 중심으로 가벼운 콘텐츠 계획을 세울 때

## 확인 사항

다음을 명확히 하십시오:

* **원본 소재**: 어떤 콘텐츠를 기반으로 가공할 것인가
* **타겟 오디언스**: 개발자, 투자자, 고객, 운영자 또는 일반 대중 등
* **플랫폼**: X, LinkedIn, TikTok, YouTube, 뉴스레터 또는 멀티 플랫폼
* **목표**: 브랜드 인지도, 전환, 채용, 권위 형성, 출시 지원 또는 단순 소통 등

## 핵심 규칙

1. **플랫폼 최적화**: 똑같은 문구를 여러 곳에 그대로 복사하지 마십시오.
2. **후킹(Hook)의 중요성**: 서론의 후킹 문구가 결론보다 훨씬 중요합니다.
3. **단일 아이디어**: 포스트 하나에는 명확한 아이디어 하나만 담으십시오.
4. **구체성**: 추상적인 구호보다는 구체적인 디테일을 사용하십시오.
5. **명확한 CTA**: 행동 유도(Call to Action)는 작고 명확하게 만드십시오.

## 플랫폼별 가이드라인

### X (트위터)
* 도입부를 빠르게 전개하십시오.
* 스레드의 각 포스트에는 하나의 아이디어만 넣으십시오.
* 꼭 필요한 경우가 아니면 본문에 링크를 넣지 마십시오.
* 해시태그를 남용하지 마십시오.

### LinkedIn
* 첫 번째 문장을 강렬하게 작성하십시오.
* 짧은 단락을 사용하십시오.
* 교훈, 결과, 시사점을 중심으로 명확하게 구조화하십시오.

### TikTok / 숏폼 영상
* 처음 3초 안에 시선을 사로잡아야 합니다.
* 내레이션뿐만 아니라 시각적 요소를 고려하여 스크립트를 짜십시오.
* 하나의 데모, 하나의 주장, 하나의 행동 유도만 담으십시오.

### YouTube
* 결과를 가능한 한 일찍 보여주십시오.
* 챕터별로 내용을 구조화하십시오.
* 20~30초마다 시각적 변화를 주십시오.

### 뉴스레터
* 관련 없는 내용을 나열하지 말고 명확한 관점을 제시하십시오.
* 섹션 제목만 봐도 내용을 알 수 있게 만드십시오.
* 도입부 단락에서 독자의 흥미를 확실히 끌어내십시오.

## 콘텐츠 재가공(Repurposing) 프로세스

기본 단계:
1. **앵커 소재 확보**: 기사, 영상, 프레젠테이션, 메모 또는 출시 문서 등
2. **원자 단위 아이디어 추출**: 3~7개의 핵심 아이디어를 뽑아냅니다.
3. **플랫폼별 변형 작성**: 각 플랫폼의 특성에 맞춰 내용을 다시 씁니다.
4. **중복 제거**: 서로 다른 플랫폼 결과물 간의 불필요한 중복을 다듬습니다.
5. **CTA 정렬**: 플랫폼의 특성에 맞춰 행동 유도 문구를 조정합니다.

## 결과물 구성

콘텐츠 캠페인 요청 시 다음을 제공합니다:
* 핵심 핵심 관점(Core angle)
* 플랫폼별 맞춤 초안
* 권장 게시 순서 (선택 사항)
* 행동 유도 문구(CTA) 변형 (선택 사항)
* 게시 전 추가로 필요한 정보

## 품질 확인 사항

제출 전 다음을 확인하십시오:
* 각 초안이 해당 플랫폼의 네이티브 스타일로 읽히는가
* 도입부 후킹 문구가 강렬하고 구체적인가
* 상투적인 홍보용 수식어가 포함되지는 않았는가
* 특별한 요청이 없는 한 플랫폼 간 문구가 중복되지 않는가
* 행동 유도 문구가 콘텐츠 및 오디언스와 잘 어울리는가
