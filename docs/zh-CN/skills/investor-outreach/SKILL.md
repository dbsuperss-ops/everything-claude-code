# 투자자 접촉 (Investor Outreach)

투자자들과의 소통은 간결하고, 개인화되어야 하며, 행동으로 옮기기 쉬워야 합니다.

## 적용 시점

* 투자자에게 콜드 메일을 보낼 때
* 지인에게 투자자 소개 요청(Intro request) 문구를 작성할 때
* 미팅 후 또는 무응답 시 후속(Follow-up) 메일을 보낼 때
* 투자 유치 과정에서 투자자 업데이트 메일을 작성할 때
* 펀드의 투자 테마나 파트너의 관심사에 맞춰 메시지를 맞춤화할 때

## 핵심 규칙 (Core Rules)

1. **모든 메시지를 개인화하십시오.** (복사 붙여넣기 금지)
2. **요청 사항을 작게 유지하십시오.** (Low friction)
3. **형용사 대신 증거(Evidence)를 사용하십시오.** (최고다, 혁신적이다 보니까 구체적 수치 제시)
4. **간결함을 유지하십시오.**
5. **누구에게나 보낼 수 있는 템플릿 메일은 절대 보내지 마십시오.**

## 콜드 메일 구성 (Structure)

1. **제목**: 짧고 구체적일 것 (예: [회사명] - [성장 지표/핵심 기술])
2. **오프닝**: 왜 '이 특정 투자자'에게 연락했는지 명시
3. **피칭(Pitch)**: 회사가 무엇을 하는지, 왜 지금인지, 어떤 성장 지표가 중요한지 서술
4. **요청(Ask)**: 구체적인 다음 단계 제안 (예: 15분 미팅)
5. **서명**: 이름, 직함, 신뢰도를 높여줄 한 문장 요약

## 개인화 소스 (Personalization Sources)

다음 중 하나 이상을 참고하여 메시지를 작성하십시오:
* 해당 투자자의 관련 포트폴리오사 (경쟁사가 아님을 확인)
* 투자자의 공개된 투자 테마, 강연, 기고문, SNS 포스트
* 상호 알고 있는 지인 (네트워킹)
* 투자자의 평소 관심 분야와 명확히 일치하는 시장/제품의 특성

*관련 배경 정보가 없는 경우, 사용자에게 질문하거나 템플릿의 어느 부분을 수정해야 하는지 안내하십시오.*

## 후속(Follow-up) 템포 가이드

기본 권장 일정:
* **0일차**: 최초 메일 발송
* **4-5일차**: 짧은 후속 메일 (새로운 데이터 포인트 1개 포함 권장)
* **10-12일차**: 마지막 후속 메일 (깔끔하게 마무리)

*사용자가 더 긴 시퀀스를 요청하지 않는 한 그 이상은 권장하지 않습니다.*

## 지인 소개 요청 (Intro Requests)

소개해주는 사람이 편하게 전달할 수 있도록 배려하십시오:
* 왜 이 소개가 적절한지 설명하십시오.
* 전달 가능한(Forwardable) 요약문을 포함하십시오.
* 요약문은 100단어 이내로 간결하게 작성하십시오.

## 미팅 후 업데이트 (Post-meeting)

포함 사항:
* 논의되었던 구체적인 항목
* 약속했던 답변이나 업데이트 자료
* 가능하다면 새로운 증거(Proof point) 1개
* 명확한 다음 단계(Next steps)

## 최종 품질 검토
메시지 전달 전 확인하십시오:
* [ ] 메시지가 충분히 개인화되었는가
* [ ] 요청 사항(Ask)이 명확한가
* [ ] 미사여구나 구걸하는 듯한 표현은 없는가
* [ ] 구체적인 증거/수치가 포함되었는가
* [ ] 분량이 간결하게 압축되었는가
