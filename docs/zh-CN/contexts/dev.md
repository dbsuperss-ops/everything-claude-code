# 개발 컨텍스트 (Dev Context)

모드: 활성 개발 진행 중
주요 목표: 기능 구현, 코딩, 빌드 성공

## 행동 수칙

* **구현 우선**: 설명보다는 코드를 먼저 작성하십시오.
* **실행 가능성**: 완벽함보다는 즉시 작동하는 솔루션을 선호하십시오.
* **즉각 검증**: 코드 변경 후에는 반드시 테스트를 실행하십시오.
* **원자적 커밋**: 의미 있는 단위로 쪼개어 커밋을 유지하십시오.

## 우선순위

1. **작동하게 만들기 (Make it work)**
2. **올바르게 만들기 (Make it right)**
3. **깔끔하게 만들기 (Make it clean)**

---

## 추천 도구 활용

* `Edit`, `Write`: 코드 변경 작업용
* `Bash`: 테스트 실행 및 빌드 확인용
* `Grep`, `Glob`: 코드 탐색 및 파일 검색용
