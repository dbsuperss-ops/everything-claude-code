## 후원의 장점

여러분의 후원은 다음을 가능하게 합니다:

* **신속한 개발** — 새로운 도구와 기능 개발에 더 많은 시간을 투자할 수 있습니다.
* **무료 기능 유지** — 프리미엄 기능을 통해 모든 사용자를 위한 무료 등급 운영이 가능해집니다.
* **강력한 지원** — 후원자 분들께는 우선적으로 답변을 제공합니다.
* **로드맵 참여** — Pro+ 등급 후원자는 새로운 기능 개발 우선순위에 투표할 수 있습니다.

## 후원 제안 자료

후원 논의 시 다음 데이터들을 활용하십시오:

* `ecc-universal` 및 `ecc-agentshield` 패키지의 npm 실시간 다운로드 지표
* 마켓플레이스를 통한 GitHub 앱 배포 현황
* 공개된 도입 신호: 별(stars), 포크(forks), 기여자 수, 출시 주기
* 플랫폼 지원 현황: Claude Code, Cursor, OpenCode, Codex 앱/CLI

지표를 추출하는 워크플로우는 [`docs/business/metrics-and-sponsorship.md`](../business/metrics-and-sponsorship.md)를 참조하십시오.
