# 루프 상태 (Loop-Status) 명령어

현재 활성화되어 실행 중인 루프의 상태를 점검하고 진행 상황을 파악합니다.

## 사용법

`/loop-status [--watch]`

* `--watch`: (선택 사항) 상태가 변할 때마다 주기적으로 내용을 갱신하여 표시합니다.

## 보고 항목

* 현재 활성화된 루프 패턴
* 진행 중인 단계 및 마지막으로 성공한 체크포인트
* 실패한 검사 항목 (있는 경우)
* 예상 시간 및 비용 소모 대비 편차
* 권장 조치 (계속 / 일시 중지 / 중단)

## 매개변수 (Arguments)

* `--watch` 선택 가능

**핵심**: 루프 상태 명령어를 통해 자율적으로 작업하는 에이전트의 현재 위치를 실시간으로 확인하고, 이상 징후가 보이면 즉시 개입할 수 있습니다.
