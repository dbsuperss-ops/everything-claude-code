# 모델 라우팅 (Model-Route) 명령어

주어진 작업의 난이도와 예산 조건에 가장 적합한 모델을 판단하여 제안합니다.

## 사용법

`/model-route [작업-설명] [--budget low|med|high]`

* `작업-설명` (선택 사항): 수행할 작업에 대한 자유 형식의 텍스트
* `--budget`: (선택 사항) 예산 수준 설정 (`low`, `med`, `high`)

## 라우팅 기준 (Heuristics)

* **Haiku**: 결정적이고 위험도가 낮은 단순 반복 작업이나 기계적인 코드 수정
* **Sonnet**: 일반적인 구현, 리팩토링 및 대부분의 코딩 작업을 위한 기본 선택
* **Opus**: 복잡한 아키텍처 설계, 심층적인 코드 리뷰, 모호한 요구 사항 분석

## 필수 출력 항목

* 권장되는 모델 이름
* 해당 모델 선택에 대한 신뢰도
* 선택한 근거 및 사유
* 첫 번째 시도가 실패하거나 부족할 경우를 대비한 **대체(Rollback) 모델**

## 매개변수 (Arguments)

* `[작업-설명]` 선택 가능
* `--budget low|med|high` 선택 가능

**핵심**: 모델 라우팅은 작업 효율을 극대화하면서도 불필요한 비용 소모를 줄여주는 '지능형 도우미'입니다. 복잡한 작업을 시작하기 전 가이드를 받으십시오.
